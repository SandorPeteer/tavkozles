# RC szűrő

Szül.: 2006.10.02.   

Aluláteresztő RC szűrő kapcs. 

*Adatok:*  

$R = 10 Ohm $   
$C = 2 uF $

*Határfrekvencia:*   
   
$fh= {1 \over (2 * Pi * R * C)} $   
$fh= {1 \over (2 * Pi * 10R * 2uF)} $   
$fh= 7957.7472 Hz $
